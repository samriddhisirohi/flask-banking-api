# Flask Banking API

A RESTful Banking Transaction System built using Flask and SQLAlchemy ORM.

This backend application allows users to create accounts and perform fund transfers with proper transaction handling (commit & rollback). The project demonstrates core backend engineering concepts including ORM mapping, relational databases, and atomic transactions.
